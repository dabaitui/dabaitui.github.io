前情提要

需要有jdk1.8，python2，android-studio，sdk，并配置环境

创建项目

npx react-native init Test001
最后是项目名，不需要提前装什么脚手架，直接输入命令就行
运行项目，到项目的文件夹终端里
yarn android 就可以启动项目了
如果没有yran就用这种方式
npx react-native run-iosios
npx react-native run-androidandroid
