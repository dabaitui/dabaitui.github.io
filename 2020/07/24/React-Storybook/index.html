安装

npx -p @storybook/cli sb init
自动下载并安装一些需要的文件
安装成功后就会出现


这些文件夹

使用

npm run storybook
可以启动服务

注意

Storybook是独立于react的，所以之前的Styled-components也要给Storybook加载一次
去.storybook文件夹下新建个preview.js

加入组件

在某一个组件的文件夹中新建个如avatar.stories.js的文件，一定要以.stories.js结尾，storybook会自动找这个结尾的文件
